# Get started

## Installation
Install `de-code-product` or `de-code-marketing` from npm:

```bash
npm i de-code-product
```

## Usages
de-code is just a CSS library something silimar to twitter bootsrtap and uikit, de-code packages includes a single CSS file which can be used by importing direclty from the node_modules.

```js
import styles from 'node_modules/de-code/css/index.css';
```
### CDN
You can import de-code css direclty from unpkg cdn
```js
<link src="https://unpkg.com/de-code-product/css/index.css" ref="stylesheet">
```
